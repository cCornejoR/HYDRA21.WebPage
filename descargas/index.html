Requisitos del Sistema

Asegúrate de que tu sistema Windows cumple con los requisitos mínimos y recomendados

COMPATIBLE SOLO CON WINDOWS
⚠️

Requisitos Mínimos

Para funcionamiento básico

🖥️

Sistema Operativo

Windows 10 (64-bit) o superior

🧠

Procesador

Intel i5 o AMD Ryzen 5 (4 núcleos)

💾

Memoria RAM

8 GB DDR4

💿

Almacenamiento

2 GB espacio libre (SSD recomendado)

🎮

Tarjeta Gráfica

DirectX 11 compatible

⭐

Requisitos Recomendados

Para rendimiento óptimo

🖥️

Sistema Operativo

Windows 11 (64-bit)

🧠

Procesador

Intel i7/i9 o AMD Ryzen 7/9 (8+ núcleos)

💾

Memoria RAM

32 GB DDR4/DDR5

💿

Almacenamiento

10 GB espacio libre (NVMe SSD)

🎮

Tarjeta Gráfica

NVIDIA RTX 3060+ o AMD RX 6600+
